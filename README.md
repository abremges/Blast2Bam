# Blast2Bam

Map sequences with NCBI blastn and output the results in SAM/BAM format.


# Compilation

```
$ make
```

# Usage

### Fastq2fasta

```bash
$ bin/fastq2fasta [options] FastQ_1 [FastQ_2] > out.fasta
```

### Blast2bam

```bash
$ bin/blast2bam [options] blast.xml ref.dict FastQ_1 [FastQ_2] > out.sam
```
